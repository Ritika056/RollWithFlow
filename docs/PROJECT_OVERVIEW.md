# RollWithFlow Project Overview

RollWithFlow is a private DJ operating system. It is designed for DJs, not passive listeners.

The long-term product combines:

- Music library management for local and referenced songs
- Discovery from future platforms such as Spotify and YouTube
- Liked Songs as an intentional saved collection
- Folders for practical organization
- Playlists for prepared DJ sets
- DJ Crates for flexible performance pools
- Mix archive for uploaded or recorded sets
- Music intelligence for BPM, key, genre, mood, energy, ratings, tags, and notes
- Daily fetch in a later phase for trending/latest music snapshots
- Player support in a later phase for local songs, mixes, previews, embeds, or source links
- LAN access so the private app can be used across trusted devices on the same Wi-Fi

Phase 1 intentionally builds only the foundation: application structure, database models, seed data, API reads, health checks, and a dark DJ-style shell.
