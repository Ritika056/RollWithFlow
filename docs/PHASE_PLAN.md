# RollWithFlow Phase Plan

## Phase 1 - Foundation

Create the frontend app, backend app, SQLite database setup, core SQLAlchemy models, API route structure, dashboard shell, sidebar navigation, health checks, seed/sample data, and documentation.

## Phase 2 - Library, Liked Songs, Folders

Add manual song creation and editing, richer song views, liked/unliked flows, folder management, song-to-folder assignment, reject/restore behavior, search/filtering, and missing metadata workflows.

## Phase 3 - Playlists, Crates, Mixes, Music Intelligence

Add playlist builder basics, DJ crate workflows, mix archive/upload foundations, tags, ratings, notes, BPM/key/genre editing, and useful search/filter surfaces.

## Phase 4 - Discovery, Spotify, YouTube, Daily Fetch

Add Spotify and YouTube discovery, manual search, trending/latest tracking, saved/rejected discovery history, and daily 6 AM fetch.

## Phase 5 - Player, LAN, Polish, Testing

Add playback/preview support where legal and technically available, LAN-ready configuration, UI polish, robust tests, backups/exports, and final hardening.
