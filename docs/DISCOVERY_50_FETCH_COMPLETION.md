# Discovery 50 Fetch

Discovery searches now default to 50 results and accept 25, 50, or 100 from the Discovery UI. Spotify requests pages of up to 50 with offsets. YouTube uses page tokens and requests pages of up to 50 until it reaches the requested limit or the provider is exhausted.

Fetch-run metadata records requested, fetched, created, duplicate, and error counts. Provider results remain metadata and source links only.

Trending, latest, famous, old classics, Hindi, Punjabi, Bollywood, EDM, House, Techno, and Hip Hop are query presets. They are search approximations because provider APIs do not offer one universal trending endpoint.
